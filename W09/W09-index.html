  2. BOM == browser object model

    alert, prompt, and confirm are originally browser functions. They can be accessed through the window

    window.navigator.userAgent will tell you info about the browser and the user divice. It can be tweaked so be wary.

    window.location.href will return the href of the current website as a string

    window.location.protocol will return https:, http:, pop2:, etc

    widonw.location.host will return the domain and the port number of the current url

    window.location.hostname will return the domain

    window.location.port will return the port number

    window.location.pathname will return the a string of the path of the domain

    window.location.search needs a ? and perameters

    window.location.hash returns any fragments

    window.location.orgin reads the url incuding the protocol

    window.location.reload() will reload the page

    window.location.assign() will let you reassign the url

    window.location.replace() will reassign the url but won't save it to history so the user can nav back

    window.location.toString() returns the whole url as a string


    window.history holds info on all pages visited in the present session

    window.history.length() returns how many pages have been visited before the current one

    window.history.go(x) sends the page back to another page on the list. 0 reloads, 1 goes forward, -1 goes to the first page


    window.open(url, title, variables such as width) opens a new window

    window.close() closes a window

    window.moveTo(x,y) moves the window to a coordinance

    window.resizeTo(width, height) resizes the window


    window.screen.width returns width

    window.screen.height returns height

    window.screen.availWidth returns width + any menus ect

    window.screen.availHeight returns height + any menus ect

    window.screen.colorDepth returns the color bit depth of the user's monotor
